# 🏏 Fab 4 Cricket Stats Dashboard

An interactive Cricket Dashboard built using HTML, CSS and JavaScript that compares the careers of the Fab Four of modern cricket.

## Features

- Career Comparison
- Test Stats
- ODI Stats
- T20 Stats
- Year-wise Runs
- Year-wise Centuries
- Interactive Charts
- Dark Mode
- Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Chart.js
